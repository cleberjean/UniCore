# UniCore
Turn sleep all extra (AP) cores from Processor, in Real Mode DOS, leaving only BSP core active.

Allow activate maximum turbo boost frequency/multiplier from Intel processors.

Was tested with MS-DOS 6.22, Windows 3.1 Standard/Enhanced Mode, Windows 98SE and DOSBox/DOSBox-X.

On DOSBox(-X) detects the emulation environment and aborts.

**DO NOT load with LOADHIGH command.**
